# Problem Statement
Calculating train fares manually becomes confusing because prices vary based on coach type,  
distance, discount category, and GST. Users need a simple tool to compute the accurate fare  
without doing multiple calculations manually.

# Scope of the Project
The scope of this project is :  
- Takes distance and coach type as input  
- Applies the chosen discount  
- Adds 5% GST  
- Displays the final fare  

# Target Users
- Students  
- Every individuals who want a quick fare estimate  
- Senior citizens as it is easy to use   

# High-Level Features
- Distance input  
- Coach selection (General, Sleeper, 3AC, 2AC, 1AC)  
- Discount categories (Student, Senior Citizen, Military, None)  
- GST calculation  
- Final fare output  
- Modules with 5 clearly defined functions
